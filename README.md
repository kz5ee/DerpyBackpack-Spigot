# DerpyBackpack-Spigot

# Description

This plugin is a Spigot backpack plugin. At this time, items are stored in MySQL database.

# Usage

/bp - Opens the backpack

permissions:
    dbp.use:
        default: false
    dbp.slots.18:
        default: false
    dbp.slots.27:
        default: false
    dbp.slots.36:
        default: false
    dbp.slots.45:
        default: false
    dbp.slots.54:
        default: false
